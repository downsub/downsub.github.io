YouTube automatic speech recognition

YouTube generates ASR transcripts of videos with speech, but doesn't make them publicly available to download without an API key.

However, you can use the YouTube Editor to create new captions from transcripts of your own videos, and download those.

 

Adding API functionality would allow anyone to download ASRs directly, but the free limit is 50 per day, so it's not something I can set up and forget about.
